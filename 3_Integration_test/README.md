# Tehtävä 3 Integration testing

## Yksikkö- ja integraatiotestauksen tehtävät expressillä toteutettua rajapintaa vastaan, jolla voidaan konvertoida värikoodeja hex-muodosta rgb-muotoon

Aja seuraavat komennot testataksesi:

1. npm init -y
2. npm install express --save
3. npm install --save-dev mocha chai supertest

Aja testit:

- npm test

Kehitysympäristön käynnistys:

- npm run dev

Dokumentoitu JSDocsilla, luodaksesi html dokumentaation aja:

1. npm install --save-dev jsdoc
2. npx jsdoc \<tiedosto>
